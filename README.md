# RHENU

RHENU is a full-stack digital reading platform for **light novels, manga, and books**, built with **Next.js, TypeScript, Supabase, PostgreSQL, PDF.js, Web Audio API, and Tailwind CSS**.  
This repository shows a preview of the platform (~95% complete). While not fully polished, the screenshots below highlight the major features and user experience.

---

## Project Status
- Core functionality complete (~95%)  
- Some refinements and polish still in progress  
- This README acts as a **visual walkthrough** of current features
